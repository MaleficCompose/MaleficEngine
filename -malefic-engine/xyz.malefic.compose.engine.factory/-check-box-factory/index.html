A factory class for creating a CheckBox composable with customizable components.

This class implements the ComposableFactory interface, allowing the creation of a CheckBox with various optional parameters such as modifier, checked state, onCheckedChange callback, and colors. It provides default values for many parameters, enabling flexible and easy setup of a CheckBox.
