A factory class for creating a Scaffold composable with customizable components.

This class implements the ComposableFactory interface, allowing the creation of a Scaffold with various optional parameters such as top bar, bottom bar, floating action button, and drawer configurations. It provides default values for many parameters, enabling flexible and easy setup of a Scaffold layout.
