A factory class for creating and rendering a Composable Text with customizable properties.

This class implements the ComposableFactory interface and provides methods to configure and render a Text composable with various attributes such as color, font size, style, alignment, and more. It includes methods to compose the text, invoke it for rendering, and apply configuration changes before rendering.
